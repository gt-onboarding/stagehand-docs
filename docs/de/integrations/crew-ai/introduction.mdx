<div id="overview">
  ## Übersicht
</div>

Diese Anleitung zeigt, wie Sie CrewAI mit Browserbase verwenden, um intelligente Agenten zu erstellen, die Webinteraktionen automatisieren. Am Ende dieser Anleitung wissen Sie, wie Sie:

- CrewAI mit dem StagehandTool einrichten
- Agenten erstellen, die mit Websites interagieren können
- Browseraufgaben mithilfe natürlicher Sprachbefehle automatisieren
- Strukturierte Daten aus Webseiten extrahieren

<div id="when-youd-use-this">
  ## Wann Sie das verwenden würden
</div>

Die CrewAI-Integration ist ideal für Szenarien, in denen Sie intelligente Webautomatisierung benötigen:

- **Rechercheautomatisierung**: Lassen Sie Agents Informationen auf mehreren Websites recherchieren
- **Datenerfassung**: Extrahieren Sie strukturierte Daten von E‑Commerce‑Websites, Jobbörsen oder Nachrichtenseiten
- **Formularautomatisierung**: Füllen Sie Formulare automatisch aus und senden Sie sie nach bestimmten Kriterien ab
- **Mehrstufige Workflows**: Führen Sie komplexe Browser-Workflows aus, die Entscheidungen erfordern

Das StagehandTool kapselt das Stagehand Python SDK, um CrewAI-Agents die Möglichkeit zu geben, einen echten Webbrowser zu steuern und mit Websites über drei Kernprimitiven zu interagieren:

1. **Act**: Führen Sie Aktionen wie Klicken, Tippen oder Navigieren aus
2. **Extract**: Extrahieren Sie strukturierte Daten von Webseiten
3. **Observe**: Identifizieren und analysieren Sie Elemente auf der Seite
