<div id="overview">
  ## Überblick
</div>

Der Browserbase MCP‑Server bringt leistungsstarke Browserautomatisierung über das Model Context Protocol (MCP) zu Claude. Aufbauend auf [Stagehand](https://docs.stagehand.dev/) bietet diese Integration KI‑gestützte Web‑Automatisierung mit natürlichsprachigen Befehlen.

<Info>
  Dieser Server ermöglicht es Claude, Browser zu steuern, Websites zu navigieren, mit Webelementen zu interagieren und Daten zu extrahieren – alles über einfache, dialogbasierte Befehle.
</Info>

<Note>
  **Zwei Möglichkeiten, MCP zu nutzen:**

  1. **Browserbase MCP‑Server**: Stellt Claude Desktop und anderen MCP‑Clients Browserautomatisierungstools bereit
  2. **MCP‑Integrationen in Stagehand**: Externe Tools (wie Exa Search, Supabase) zu Stagehand‑Agents hinzufügen

  Diese Dokumentation behandelt den Browserbase MCP‑Server. Informationen zur Verwendung von MCP‑Integrationen innerhalb von Stagehand‑Agents finden Sie unter [MCP Integrations](/de/best-practices/mcp-integrations).
</Note>

<div id="session-management">
  ## Sitzungsverwaltung
</div>

<Info>
  Der Browserbase MCP Server unterstützt sowohl Einzel- als auch Multi‑Sitzungsarchitekturen, um unterschiedliche Automatisierungsanforderungen abzudecken.
</Info>

<Tabs>
  <Tab title="Single Session Mode">
    **Traditioneller Ansatz**

    - Eine aktive Browsersitzung zur gleichen Zeit
    - Einfacher für grundlegende Automatisierungsaufgaben
    - Automatisches Sitzungs‑Lifecycle‑Management
    - Ideal für sequentielle Workflows
  </Tab>

  <Tab title="Multi-Session Mode">
    **Erweiterte parallele Verarbeitung**

    - Mehrere unabhängige Browsersitzungen
    - Jede Sitzung verwaltet ihren eigenen Zustand
    - Parallele Ausführung
    - Perfekt für komplexe Workflows
  </Tab>
</Tabs>

<div id="getting-started">
  ## Erste Schritte
</div>

<Steps>
  <Step title="MCP-Server installieren">
    Wählen Sie je nach Bedarf zwischen NPM-Installation, gehosteter Remote-URL oder lokaler Entwicklung.
  </Step>

  <Step title="Authentifizierung konfigurieren">
    Hinterlegen Sie Ihre Browserbase-API-Zugangsdaten in der MCP-Konfiguration.
    Ihre API-Schlüssel erhalten Sie im [Browserbase-Dashboard](https://www.browserbase.com/overview).
  </Step>

  <Step title="Automatisierung starten">
    Steuern Sie Browser mit natürlichen Sprachbefehlen über Claude.
  </Step>
</Steps>

<Tip>
  Bereit loszulegen? Sehen Sie sich unsere [Setup-Anleitung](/de/integrations/mcp/setup) für detaillierte Installationsinstruktionen an.
</Tip>
