Stagehand basiert auf [Playwright](https://playwright.dev/), daher können Sie Playwright-Methoden direkt über die Stagehand-Instanz aufrufen.

<div id="page-and-context">
  ## `page` und `context`
</div>

`stagehand.page` und `stagehand.context` sind Instanzen von Playwrights `Page` bzw. `BrowserContext`. Verwenden Sie diese, um mit der Playwright-Instanz zu interagieren, die Stagehand nutzt.

<CodeGroup>
```TypeScript TypeScript
const page = stagehand.page;
// Grundlegende Playwright-Methoden funktionieren
await page.goto("https://github.com/browserbase/stagehand");

// Stagehand überschreibt Playwright-Objekte
await page.act("click on the contributors")
```

```python Python
page = stagehand.page
# Grundlegende Playwright-Methoden funktionieren
await page.goto("https://github.com/browserbase/stagehand")

# Stagehand überschreibt Playwright-Objekte
await page.act("click on the contributors")
```
</CodeGroup>
