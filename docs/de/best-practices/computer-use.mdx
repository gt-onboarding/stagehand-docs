</div>

Vielleicht haben Sie schon von [Claude Computer Use](https://www.anthropic.com/news/3-5-models-and-computer-use) oder dem [Computer Using Agent von OpenAI](https://openai.com/index/computer-using-agent/) gehört.

Diese leistungsfähigen Tools können natürliche Sprache in Aktionen auf dem Computer umsetzen. Ohne sie müssten Sie eigenen Code schreiben, um diese Aktionen in Playwright-Befehle zu überführen.

Stagehand übernimmt nicht nur die Ausführung der Computer-Use-Ergebnisse, sondern ermöglicht auch das nahtlose Umschalten zwischen OpenAI- und Anthropic-Modellen mit nur einer einzigen Codezeile.

<div id="how-to-use-a-computer-use-agent-in-stagehand">
  ## Verwendung eines Computer-Use-Agents in Stagehand
</div>

Mit Stagehand kannst du Computer-Use-Agents mit nur einer Codezeile einsetzen:

<Note>
**WICHTIG! Browserabmessungen konfigurieren**

Computer-Use-Agents liefern häufig XY-Koordinaten zum Klicken auf dem Bildschirm, daher musst du die Browserabmessungen festlegen.

Wenn nichts angegeben ist, beträgt die Standardgröße 1024x768. Alternativ kannst du die Browserabmessungen über die Optionen `browserbaseSessionCreateParams` oder `localBrowserLaunchOptions` konfigurieren.
</Note>

<div id="configuring-browser-dimensions">
  ### Browserabmessungen konfigurieren
</div>

Die Browserkonfiguration variiert je nach Umgebung:

<Tabs>
<Tab title="BROWSERBASE">
<CodeGroup>  
```typescript TypeScript
import { Stagehand } from "@browserbasehq/stagehand";

const stagehand = new Stagehand({
	env: "BROWSERBASE",
  	apiKey: process.env.BROWSERBASE_API_KEY /* API-Schlüssel zur Authentifizierung */,
    projectId: process.env.BROWSERBASE_PROJECT_ID /* Projekt-ID */,
  
    browserbaseSessionCreateParams: {
      projectId: process.env.BROWSERBASE_PROJECT_ID!,
      browserSettings: {
		blockAds: true,
        viewport: {
          width: 1024,
          height: 768,
        },
      },
  	},
});

await stagehand.init();
```
```python Python
import os
from stagehand import Stagehand, StagehandConfig

stagehand = Stagehand(StagehandConfig(
    env="BROWSERBASE",
    api_key=os.getenv("BROWSERBASE_API_KEY"),  # API-Schlüssel zur Authentifizierung
    project_id=os.getenv("BROWSERBASE_PROJECT_ID"),  # Projekt-ID
    
    browserbase_session_create_params={
        "projectId": os.getenv("BROWSERBASE_PROJECT_ID"),
        "browserSettings": {
            "blockAds": True,
            "viewport": {
                "width": 1024,
                "height": 768,
            },
        },
    },
))

await stagehand.init()
```

<div id="direct-your-computer-use-agent">
  ### Steuern Sie Ihren Computer-Use-Agent
</div>

Rufen Sie `execute` auf dem Agenten auf, um ihm eine Aufgabe zuzuweisen.

<CodeGroup>
```typescript TypeScript
// Zu einer Website navigieren
await stagehand.page.goto("https://www.google.com");

const agent = stagehand.agent({
	// Sie können entweder OpenAI oder Anthropic verwenden
	provider: "anthropic",
	// Das zu verwendende Modell (computer-use-preview für OpenAI)
	model: "claude-sonnet-4-20250514",

	// System-Prompt anpassen
	instructions: `You are a helpful assistant that can use a web browser.
	Do not ask follow up questions, the user will trust your judgement.`,

	// API-Schlüssel konfigurieren
	options: {
		apiKey: process.env.ANTHROPIC_API_KEY,
	},
});

// Agenten ausführen
await agent.execute("Apply for a library card at the San Francisco Public Library");
```

```python Python
import os

# Zu einer Website navigieren
await stagehand.page.goto("https://www.google.com")

agent = stagehand.agent({
    # Das zu verwendende Modell
    model="computer-use-preview",

    # System-Prompt anpassen
    instructions="You are a helpful assistant that can use a web browser. Do not ask follow up questions, the user will trust your judgement.",

    # API-Schlüssel konfigurieren
    options={
        "apiKey": os.getenv("ANTHROPIC_API_KEY"),
    },
})

# Agenten ausführen
await agent.execute("Apply for a library card at the San Francisco Public Library")
```
</CodeGroup>

Sie können außerdem die maximale Anzahl an Schritten festlegen, die der Agent ausführen darf, mit:

<CodeGroup>
```typescript TypeScript
await agent.execute({
	instructions: "Apply for a library card at the San Francisco Public Library",
	maxSteps: 10,
});
```

```python Python
await agent.execute(
	"Apply for a library card at the San Francisco Public Library",
	max_steps=10,
)
```
</CodeGroup>
