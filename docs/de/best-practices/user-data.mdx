<div id="user-data-directory">
  ### Benutzerdatenverzeichnis
</div>

Bewahren Sie Browserdaten über Sitzungen hinweg auf, indem Sie ein eigenes Benutzerdatenverzeichnis verwenden:

<CodeGroup>
```typescript TypeScript
import { Stagehand } from "@browserbasehq/stagehand";

// Für Browserbase-Sitzungen
const stagehand = new Stagehand({
  env: "BROWSERBASE",
  browserbaseSessionCreateParams: {
    userDataDir: "/path/to/user/data/directory",
  },
});

// Für lokale Sitzungen
const localStagehand = new Stagehand({
  env: "LOCAL",
  localBrowserLaunchOptions: {
    userDataDir: "./browser-data",
  },
});

await stagehand.init();
console.log("Session ID:", stagehand.sessionId);
```
```python Python
from stagehand import Stagehand

# Für Browserbase-Sitzungen
stagehand = Stagehand(
    env="BROWSERBASE",
    browserbase_session_create_params={
        "user_data_dir": "/path/to/user/data/directory",
    },
)

# Für lokale Sitzungen
local_stagehand = Stagehand(
    env="LOCAL",
    local_browser_launch_options={
        "user_data_dir": "./browser-data",
    },
)

await stagehand.init()
print(f"Session ID: {stagehand.session_id}")
```
