Stagehand ist ein Browser-Automatisierungsframework, mit dem sich Webbrowser per natürlicher Sprache und Code steuern lassen. Durch die Kombination der Leistungsfähigkeit von KI mit der Präzision von Code macht Stagehand Webautomatisierung flexibel, wartbar und wirklich zuverlässig.

<div id="the-problem-with-browser-automation">
  ## Das Problem mit der Browser-Automatisierung
</div>

Traditionelle Frameworks wie Playwright und Puppeteer zwingen dich, fragile Skripte zu schreiben, die bei jeder UI-Änderung zerbrechen. Web-Agents versprechen, das mit KI zu lösen, liefern dich aber unberechenbarem Verhalten aus.

**Du steckst zwischen zwei schlechten Optionen fest:**

- **Zu fragil**: Klassische Selektoren brechen, sobald sich Websites ändern
- **Zu agentisch**: KI-Agents sind unvorhersehbar und kaum zu debuggen

<div id="enter-stagehand">
  ## Lernen Sie Stagehand kennen
</div>

Stagehand verbindet das Beste aus beiden Welten mit vier leistungsstarken Primitiven, mit denen Sie exakt steuern, wie viel KI Sie einsetzen:

<CodeGroup>
  ```typescript TypeScript
  // Act - Execute natural language actions
  await page.act("click the login button");

  // Extract - Pull structured data
  const { price } = await page.extract({
    schema: z.object({ price: z.number() })
  });

  // Observe - Discover available actions
  const actions = await page.observe("find submit buttons");

  // Agent - Automate entire workflows
  const agent = stagehand.agent({
      provider: "anthropic",
      model: "claude-sonnet-4-20250514",
      options: {
        apiKey: process.env.ANTHROPIC_API_KEY,
      },
  })
  await agent.execute("apply for this job");
  ```

  ```python Python
  # Act - Execute natural language actions
  await page.act("click the login button")

  # Extract - Pull structured data
  result = await page.extract(
    schema={"price": float}
  )

  # Observe - Discover available actions
  actions = await page.observe("find submit buttons")

  # Agent - Automate entire workflows
  await agent.execute("apply for this job")
  ```
</CodeGroup>

<div id="why-developers-choose-stagehand">
  ## Warum Entwickler sich für Stagehand entscheiden
</div>

- **Präzise Kontrolle**: Kombinieren Sie KI‑gestützte Aktionen mit deterministischem Code. Sie entscheiden genau, wie viel KI Sie einsetzen.

- **Wirklich reproduzierbar**: Aktionen exakt aufzeichnen und wiedergeben. Kein „Bei mir hat’s funktioniert“ mehr bei Browser‑Automatisierungen.

- **Wartbar im großen Maßstab**: Ein Skript kann mehrere Websites automatisieren. Wenn sich Websites ändern, passen sich Ihre Automatisierungen an.

- **Komponierbare Tools**: Wählen Sie Ihren Automatisierungsgrad mit act(), extract(), observe() und agent().
